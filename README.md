# 🏠 Inmobiliaria Remolina

Aplicación web para la gestión y visualización de propiedades inmobiliarias.  
Permite consultar propiedades, visualizar detalles y consumir datos desde una API construida con Django REST Framework.

---

## 🛠 Tecnologías utilizadas

- **Backend:** Django + Django REST Framework
- **Frontend:** HTML, CSS, JavaScript
- **Base de datos:** SQLite
- **Control de versiones:** Git & GitHub

---

## ⚙️ Funcionalidades principales

- 📌 Listado de propiedades
- 🔍 Visualización de detalle de cada propiedad
- 🌐 Consumo dinámico de API REST
- 🎨 Interfaz web responsive
- 🔐 (En desarrollo) Autenticación de usuarios
