🏡 Inmobiliaria Remolina

📌 Descripción del proyecto

Inmobiliaria Remolina es una página web diseñada para mostrar propiedades inmobiliarias de forma clara, visual y atractiva para los usuarios.

El objetivo principal del proyecto es crear una plataforma simple y moderna donde los clientes puedan:

visualizar propiedades disponibles

conocer precios

ver detalles de cada inmueble

contactar fácilmente con la inmobiliaria

Este proyecto fue desarrollado como práctica de desarrollo web utilizando HTML, CSS y GitHub Pages para publicar el sitio en internet.

🌐 Sitio Web

Puedes ver el proyecto publicado en:

👉 https://danielremolina-pro.github.io

🎯 Objetivo del proyecto

El propósito de esta página es simular el funcionamiento básico de una web inmobiliaria profesional, donde se presentan propiedades destacadas con imágenes, precios y descripciones.

Además, el proyecto busca aprender y aplicar conceptos como:

estructura de páginas web

diseño visual

organización de contenido

publicación de sitios web

uso de GitHub para hosting

🛠 Tecnologías utilizadas

El proyecto fue desarrollado utilizando tecnologías básicas de desarrollo web:

HTML5 → estructura de la página

CSS3 → diseño y estilos visuales

GitHub Pages → publicación del sitio web

Visual Studio Code → editor de código

🧱 Estructura de la página

La página está dividida en 4 secciones principales.

1️⃣ Barra de navegación (Navbar)

La parte superior de la página contiene:

nombre de la empresa Inmobiliaria Remolina

menú de navegación

Opciones del menú:

Inicio

Propiedades

Contacto

Intención

Permitir que el usuario se desplace fácilmente por la página.

2️⃣ Sección principal (Hero)

Esta es la primera sección visible de la página.

Contiene:

imagen de fondo de una casa moderna

título principal

Encuentra tu hogar ideal

subtítulo descriptivo

botón "Ver propiedades"

Intención

Captar la atención del visitante y comunicar inmediatamente el objetivo de la página: encontrar una propiedad ideal.

3️⃣ Propiedades destacadas

Esta sección muestra tres propiedades inmobiliarias con la siguiente información:

🏡 Villa moderna

Precio: $2.000.000.000

Descripción: villa con piscina y jardín.

🏢 Apartamento de lujo

Precio: $700.000.000

Descripción: apartamento con excelente vista.

🌳 Finca privada

Precio: $1.300.000.000

Descripción: finca ideal para cultivo o ganadería.

Cada propiedad incluye:

imagen

nombre

precio

descripción

botón "Ver detalles"

Intención

Permitir que el usuario visualice rápidamente las opciones disponibles y pueda explorar propiedades.

4️⃣ Formulario de contacto

La última sección de la página incluye un formulario donde el usuario puede enviar un mensaje.

Campos:

Nombre

Email

Mensaje

Botón:

Enviar
Intención

Permitir que los posibles clientes se comuniquen con la inmobiliaria para solicitar más información.
